## How this library is organized

Each outing folder is named `YYYY-MM-DD - Region - Place` and contains a `Photos\` (and sometimes `Videos\`) subfolder, already run through what looks like an AA culling pass:

- **`AA_Tier1_Best\`** - top picks, subdivided `aerial\` / `ground\`. Only present for **one** outing in this archive: 2023-08-14 Zermatt. Everywhere else, Tier 1 either was not run or was already pulled out of this folder.
- **`AA_Tier2_Alternates\`** - solid alternates/runners-up. Present for almost every outing. This is the main pool of usable, already-screened shots in this archive.
- **`AA_Tier3_Maybe\`** (often with a `people\` or, for Zermatt only, `harsh-light\` / `landmark-property\` / `soft-or-crop\` subfolder) - marginal / needs-a-second-look shots.
- **`Personal\`**, **`Social\`**, **`Trash\`** - non-commercial, social-only, or discard. Excluded from shortlists.
- **`_masters\`** - a parallel tree mirroring the tiers above, holding the higher-resolution/original master version of each selected file (same filenames). A few show explicit `_optimized`/`_exported` suffixes (drone JPEGs), confirming a real export pass happened.
- **`Drone\`** (Ayent only) - raw drone captures kept separate from the ground-camera tiering.

Three top-level folders sit outside the dated-outing structure:

- **`_DRONE_STILLS\`** / **`_DRONE_STILLS_masters\`** - drone video frame-grabs (filenames like `DJI_0002_f13s.jpg`, i.e. frame at 13s), grouped by outing date/place. Only 3 outings represented (2023-08-01 Unknown, 2023-08-05 Glarus Sud, 2023-08-06 Innertkirchen).
- **`_DRONE_VIDEO\`** - split into `AA_ready\` (explicitly finished, stock-ready drone video exports - the strongest video candidates in the whole archive) and `Social_ready\` (smaller/lighter exports meant for social, lower commercial priority). `Social_ready\` also reveals outings that do not otherwise appear as dated Photos folders: Italy - Cassina de Pecchi, Italy - Sona, Italy - Torno, Colorado - Denver (2023-09-13), and a second Colorado - Boulder County pass (2023-10-08).
- **`No location\`** - 41 dated `Unknown-location` subfolders (geocoding failed) plus a handful of `Videos\` folders, totaling 249 files / 28269.4 MB. Worth a manual location pass before Adventure-page use.
