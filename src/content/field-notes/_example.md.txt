Write your story here, in plain text.

Leave a blank line between paragraphs, and they'll render as separate
paragraphs. That's the whole trick — no formatting knowledge required.

## You can add a section heading like this

Start a line with "## " for a sub-heading. Start a line with "- " for a
bullet point:

- one thing
- another thing

To make a word **bold**, wrap it in two asterisks. To make it *italic*, use
one. To add a link, write it like [this](https://example.com).

When you're happy with it, save the file. If you set `draft` to `true` above,
switch it to `false` when you're ready for the world to see it.
